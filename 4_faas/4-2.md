1. API GatewayがFunctionsを実行できるようにポリシー設定を行う。
   
`Allow any-user to use functions-family in compartment <Compartment>`

2. API Gatewayを作成する。
   ![](./apiGateway01.png)

3. デプロイメントを作成する。
